# 一个简陋的支持添加，修改，删除栏目与子栏目的导航栏

## 功能实现逻辑参考 vue 经典案例 todoList

<br>

![导航栏样示](https://github.com/XuJiazhi/ArjarinNav/blob/main/img_readme/navBarLookLike.PNG)

本导航栏组件基于 element-ui 自带的导航栏进一步开发，so：

```
npm install element-ui
```
